### Computational Geometry & Algorithms
This repository contains implementations of fundamental algorithms focused on mathematical logic and computational efficiency. The projects explore optimized solutions for complex geometric problems and data searching.

### Key Implementations:

* **Jarvis March (Convex Hull)**: An algorithm for finding the smallest convex polygon that encloses a set of points.
* **3SUM Problem**: An optimized approach to finding unique triplets in a dataset, focusing on reducing time complexity to $O(n^2)$.
* **PSLG (Planar Straight-Line Graphs)**: Data structures designed for modeling and navigating planar subdivisions.
* **Geometric Predicates (A2D, LIE, LT)**: Algorithms for line intersections, point positioning, and spatial relationships.
